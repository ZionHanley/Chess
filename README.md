# Chess

### Overview
A chess game with the main focus being around an AI cpu for the user to play against. Other features include a score tracking system and the ability to quick save and resume easily.

### Problem Statement
    - A chess game and a chess engine program that utillizes AI to put plainly the resualts and the reasonings that can be taken from the engine.

### Key Features
    - Chess engine
    - Chess chat bot powered with AI
